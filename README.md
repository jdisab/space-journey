# KidTalk Space Journey

KidTalk Space Journey is a browser-based therapeutic conversation game for children 10 and under.

The child becomes an astronaut and answers 100 simple, space-themed questions while moving through a mission from dream to launch. The questions help therapists, counselors, school clinicians, and other helping professionals get to know a child in a playful way.

## What is new in this version

This version adds a bigger and more exciting setup screen:

- **24 avatar options**
  - male astronauts across skin tones
  - female astronauts across skin tones
  - additional astronaut-style options
  - fun characters like aliens, robots, and animal explorers
  - avatar options are shown without names/labels so kids can simply pick what looks fun
- **24 space suit options**
- larger, easier-to-see avatar buttons
- the first two mission questions are now:
  1. **Choose a name for your spaceship**
  2. **What do you want your spaceship to look like?**

This version also keeps:
- chapter complete screens
- milestone newspapers
- final newspaper article
- more reliable epic fantasy/adventure chapter-based music
- a 100-question conversation deck

## How to run it

Open `index.html` in a web browser.

## How to upload to GitHub Pages

1. Put all files in the root of your repository.
2. Go to **Settings** in the repository.
3. Open **Pages**.
4. Set:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Save.

Then your site should be live at:

```text
https://jdisab.github.io/space-journey/
```

## Project purpose

This is a rapport-building and conversation-support tool. It is not a diagnostic or formal assessment tool.

The question deck is designed to help a therapist learn about:
- feelings
- self-esteem
- home and family
- relationships
- communication
- regulation
- interests and strengths
- safety and boundaries
- hopes and future thinking
